A estrutura do html é formato pela tag doctype, este indica que o arquivo é do tipo html. A tag html é um elemento raiz de toda pagina. Para continuar a estrutura temos a tag head que é o cabeçalho com metadados, dentro dessa tag temos a tag title que indica o titulo da aba do navegador, outras tags como tambem a tag style que estiliza o texto. Em seguida, temos a tag body no qual dentro dele é indicada ao corpo da pagina, ou melhor ao conteudo do arquivo

A diferenca entre a internet e a web, a internet é a parte fisica na qual cria uma rede de computadores que estabelezem a comunicação, porem sem nunhum tipo de conteudo em quanto a web tem arquivos/documentos com o conteudo, porem essa se utiliza da internet para ser divulgada.
