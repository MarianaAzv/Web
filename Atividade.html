O que é JavaScript?

a) A motivação para a criação do JavaScript surgiu devido à natureza estática das páginas da web nos seus primeiros anos. Os usuários não tinham como interagir com os sites, o que tornava a experiência de navegação limitada, pouco envolvente e, muitas vezes, demorada. O JavaScript foi desenvolvido para trazer interatividade e dinamismo às páginas, permitindo uma navegação mais atrativa e responsiva.

b) O código JavaScript pode ser executado tanto no navegador quanto no servidor, mas suas funções são distintas em cada ambiente. No navegador (client-side), o foco está em melhorar a interação e o comportamento da página, como manipulação de elementos da interface, criação de layouts dinâmicos, animações, navegação entre seções e validação de formulários.
Já no servidor (server-side), o código é responsável por processar dados, acessar e manipular bancos de dados, validar informações recebidas, gerenciar autenticações e enviar ao navegador apenas os dados necessários. Ou seja, enquanto o código do cliente cuida da interação visual, o código do servidor gerencia a lógica e o armazenamento de dados.

c.1)document: Permite acessar e manipular o conteúdo da página web.

c.2)console:Usado para registrar mensagens no console do navegador, útil para depuração.

c.3)alert: Exibe caixas de diálogo simples para o usuário, geralmente com mensagens de alerta ou informações.
